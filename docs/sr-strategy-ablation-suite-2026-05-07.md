## 直接结论

- `70k / 80k / 90k` 必须作为独立梯度点看，不应只用 `50k / 100k` 代表。
- `只看税率` 是完全不同策略：它取消了大户榜单确认，也取消了 FDV 低于榜单成本的安全垫，容易更早进入，但风险含义不同。
- 真正可比较的口径应分成：单变量、取消变量、固定变量梯度、多变量组合。
